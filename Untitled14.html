Question 1 The total number of multiplications performed is nkm, and the total number of additions performed is nm(k-1). For one element of the matrix, the multiplication happens k times, and as the number of elements in the resultant matrix is nm the total number of multiplication performed is nkm. Same with the additions, for one element of the matrix, the addiction happened (k-1) times, and as the number of elements in the resultant matrix is nm, the total additions performed is nm(k-1).

list1=[[2,3,4],[1,5,6],[3,6,4]]
list2=[[2,4],[4,5],[5,6]]
def multiplication(a,b):
    result = [[0 for i in range(len(b[0]))] for i in range(len(a))]
    for i in range(len(a)):
        for j in range(len(b[0])):
            for k in range(len(b)):
                result[i][j] += a[i][k] * b[k][j]
    return result
import numpy as np
x=(np.dot(list1,list2))
print(x)
print(multiplication(list1,list2))
